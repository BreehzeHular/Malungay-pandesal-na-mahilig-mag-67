# 🎰 Lucky Spin 4 – Arcade 6767

## 📌 Project Overview
Lucky Spin 4 is a simple arcade-style mini-game where users spin four slots.  
If the result matches the winning combination **6-7-6-7**, the player receives a jackpot reward recorded on the blockchain using Stellar Soroban smart contracts.

---

## ❗ Problem It Solves
Many student blockchain projects are too complex or not interactive.  
This project solves that by creating a **simple, fun, and demo-able game** that clearly shows how blockchain can be used for **transparent rewards and transactions**.

---

## ⚙️ How It Works
1. The user clicks the **Spin** button  
2. Four animated slots randomly generate numbers (0–9)  
3. The result is displayed (e.g., 3-7-1-9)  
4. The system checks the result:  
   - If **6767 → Jackpot 🎉**  
   - Otherwise → Try again  
5. If the user wins:  
   - A reward is triggered  
   - The result is recorded on-chain using a Soroban smart contract  

---

## 🧩 Features
- 🎮 Simple arcade UI (slot machine style)  
- 🔁 Randomized spin system  
- 🏆 Jackpot detection (6767 combination)  
- 🔗 Blockchain reward system (Stellar Soroban)  
- 📜 Transparent result logging  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Logic: JavaScript  
- Blockchain: Stellar + Soroban Smart Contracts
